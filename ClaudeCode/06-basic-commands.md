# 06. 기본 명령어

자주 쓰는 내장 슬래시 명령어 모음이다.

전체 명령어는 [공식 문서](https://code.claude.com/docs/ko/commands)에서 확인할 수 있다.

## 세션 관리

| 명령어 | 설명 | 용도 |
|---|---|---|
| `/clear` | 대화 기록을 지운다 | 새 주제를 시작하거나 컨텍스트가 너무 길어졌을 때 |
| `/compact` | 대화를 요약해서 컨텍스트를 줄인다 | 토큰이 많이 쌓였지만 흐름은 이어가고 싶을 때 |
| `/resume` | 이전 세션을 이어서 시작한다 | 대화를 끊었다가 다시 이어갈 때 |
| `/exit` | Claude Code를 종료한다 | |

## 설정·환경

| 명령어 | 설명 | 용도 |
|---|---|---|
| `/model` | 사용할 모델을 바꾼다 | 작업 성격에 따라 속도와 성능을 조절할 때 |
| `/fast` | Fast 모드를 켜고 끈다 | 같은 모델에서 더 빠른 출력이 필요할 때 |

## 정보 확인

| 명령어 | 설명 | 용도 |
|---|---|---|
| `/cost` | 현재 세션의 토큰 사용량과 비용을 확인한다 | 세션이 길어졌을 때 사용량 점검 |
| `/status` | 계정과 구독 상태를 확인한다 | |
| `/help` | 사용 가능한 명령어 목록을 본다 | |

## 코드 작업

| 명령어 | 설명 | 용도 |
|---|---|---|
| `/review` | 현재 변경사항을 코드 리뷰한다 | 커밋 전 빠르게 검토할 때 |
| `/pr-comments` | PR 코멘트를 불러온다 | PR 리뷰 내용을 확인하고 반영할 때 |

## 프로젝트

| 명령어 | 설명 | 용도 |
|---|---|---|
| `/init` | 프로젝트 루트에 CLAUDE.md를 생성한다 | 새 프로젝트에 처음 설정할 때 |
| `/mcp` | 연결된 MCP 서버 목록과 상태를 확인한다 | MCP 연결 문제를 점검할 때 |
| `/plugins` | 현재 활성화된 플러그인 목록을 확인한다 | |

## 계정·기타

| 명령어 | 설명 | 용도 |
|---|---|---|
| `/login` | 계정을 전환한다 | 여러 계정을 쓸 때 |
| `/logout` | 로그아웃한다 | |
| `/doctor` | Claude Code 설치 상태를 점검한다 | 동작이 이상할 때 먼저 실행 |
