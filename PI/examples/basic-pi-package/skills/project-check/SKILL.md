# Project Check

## 절차

1. 프로젝트 루트의 주요 파일을 확인한다.
2. README, package.json, build.gradle, pom.xml 등 실행 명령이 담긴 파일을 찾는다.
3. 테스트, 빌드, lint 명령 후보를 정리한다.
4. 명령을 실행하기 전에는 사용자에게 실행 여부를 확인한다.

## 출력

- 기술 스택 후보
- 주요 디렉터리
- 실행 명령 후보
- 검증 명령 후보
- 먼저 읽을 파일
